## 1. Backend Engineering

### Spring Boot / Java
- **Keycloak 인증 흐름 분석**
  - PKCE 기반 OAuth2 Authorization Code Flow 이해
  - Access Token 만료로 인한 세션 끊김 현상 원인 진단
  - BFF(Backend for Frontend) 세션 패턴 설계 및 디버깅 (`BffSessionState`, `BffSessionAuthenticationFilter`)

- **동시성 제어**
  - `synchronized(session)` + 더블체크 락킹(Double-Check Locking) 패턴 설계
  - 세션 단위 락을 통한 concurrent refresh 요청 중복 방지 구조 검토

- **테스트 코드 작성**
  - `MockRestServiceServer.bindTo(RestClient.Builder)` 기반 HTTP 모킹 (Spring Boot 4.x)
  - Mockito `@MockitoSettings(strictness = Strictness.LENIENT)` 활용
  - 성공 / 4xx 실패 / 5xx 장애 / null 세션 경계 케이스 단위 테스트 설계

## 2. Frontend Engineering

### Phaser.js (게임 클라이언트)
- **Scene 생명주기 이해**
  - `create()` / `shutdown()` / `scene.restart()` 동작 원리 파악
  - 씬 재시작 시 모듈 레벨 싱글톤 상태 유지/초기화 구분
  - `registry`를 통한 씬 간 데이터 전달 패턴 활용

- **비동기 로딩 디버깅**
  - `fire-and-forget` vs `await` 패턴 적용 시 발생하는 레이스 컨디션 분석
  - 주차별 대화 데이터 로드 타이밍 불일치(Week mismatch) 원인 추적 및 수정
  - 중복 로드 요청 방지 로직 강화 (`lastRequestedWeek` 조건 추가)

- **렌더링 제어**
  - `setDepth()`를 통한 오브젝트 렌더링 순서 제어
  - `Tween` 체이닝을 활용한 연속 애니메이션 구현
  - `camera.flash()` / `camera.shake()` 이펙트 연출

- **미니게임 구현 (LottoScene)**
  - 번호 선택 상태 관리 (`selectedNumbers: number[]`)
  - 버튼 활성/비활성(disabled) 상태 동적 처리
  - 등수별 차별화 이펙트 연출 (1등/2~4등/5등/꽝 각각 분기)
  - 당첨 영역 잠금/해제 애니메이션 (`setAlpha` + Tween 페이드)
  - 자동선택, 초기화, 구매 흐름 UX 설계

### TypeScript
- 모달 dismiss 콜백 패턴 구현 (`dismissActivityModal`)
- 다이얼로그 폴백 체인 설계 (`_w{hudWeek}` → `_w{loadedWeek}` → 기본 대화)
- 타입 명시 및 옵셔널 체이닝 활용

## 3. DevOps / 협업

### Git / GitLab
- Merge conflict 분석 및 원격 버전 기준 해결
- Protected branch 정책 이해, MR(Merge Request) 기반 배포 플로우 운영
- 커밋 단위 문제 추적 (`git show --stat`, `git log --oneline`)

### CI/CD (Jenkins)
- 빌드 실패 원인 추적 및 수정
  - 리버트된 로직을 테스트하는 스테일 테스트 파일 → 컴파일 에러 → 배포 실패 흐름 분석
- Jenkins → GitLab webhook → n8n 배포 파이프라인 흐름 이해
- Blue-Green 배포 구조 이해 (`api-blue` / `api-green`)
