# TIL: 포팅(Porting)과 JNI의 이해

**학습 일자**: 2026.02.26

## 1. 포팅(Porting)이란?
* **개념**: 특정 환경(예: A 플랫폼이나 언어)에 맞춰 개발된 소프트웨어를 다른 환경(예: B 플랫폼)에서도 정상적으로 실행되도록 수정하고 이식하는 과정입니다.
* **대표적인 사례**:
  * 윈도우(Windows) 전용 소프트웨어를 리눅스(Linux) 운영체제용으로 변환하는 작업
  * 안드로이드 전용 앱을 애플 아이폰(iOS) 기기에서도 돌아가게 만드는 작업
* **크로스 플랫폼 도구**: 최근에는 플러터(Flutter)나 리액트 네이티브(React Native)처럼 코드를 한 번만 작성해도 양쪽 운영체제 모두에서 작동하도록 돕는 프레임워크가 널리 쓰이고 있습니다.
* **게임 엔진 활용**: 게임 개발 분야에서는 코코스2d(cocos2d) 같은 엔진이 이러한 플랫폼 이식 작업을 강력하게 지원합니다.

## 2. JNI (Java Native Interface)의 개념
* **정의**: C언어나 C++로 작성된 코드를 자바(Java) 프로그램 내에서 직접 실행하고 통신할 수 있게 연결해주는 규약(인터페이스)입니다.
* **동작 원리**: 정해진 함수 형태(인터페이스)를 규칙에 맞게 제공해주면, 완전히 다른 프로그래밍 언어로 짜여진 기능이라도 자바 언어의 일부인 것처럼 자연스럽게 불러와서 사용할 수 있습니다.
* **핵심 장점**: 껍데기(인터페이스)만 잘 설계하여 추상화해두면, 내부 로직이 어떤 언어로 구현되었는지에 구애받지 않고 서로 연동하며 호환성을 높일 수 있습니다.
