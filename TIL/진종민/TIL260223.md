---

## Git 컨벤션(초안 정리)
### 브랜치 네이밍
- `feat/<scope>-<short-desc>` : 기능
- `fix/<scope>-<short-desc>` : 버그 수정
- `refactor/<scope>-<short-desc>` : 리팩터링
- `chore/<scope>-<short-desc>` : 설정/잡일
- `docs/<scope>-<short-desc>` : 문서
- `remove/<scope>-<short-desc>` : 기능 제거

예)
- `feat/dialog-choice-ui`
- `fix/save-load-slot`
- `chore/eslint-config`

### 커밋 메시지
- `feat:` / `fix:` / `refactor:` / `chore:` / `docs:` / `remove:`
- 커밋은 “한 가지 변경 목적” 기준으로 쪼개기(리뷰/리버트 쉬움)

### PR/MR 운영
- PR 제목에 Jira 키 연결(예: `E203-12 feat: dialog choice ui`)
- PR 템플릿: 변경 요약 / 스크린샷 / 테스트 방법 / 영향 범위 체크리스트

---

## Jira 컨벤션(미연시 프로젝트 기준으로 재구성)
### 에픽(Epic) 분류(초안)
- `[Core] 대화/분기 시스템`
- `[Scenario] 시나리오/대사/이벤트`
- `[UI] 대화 UI & 선택지 UI`
- `[Character] 캐릭터/호감도/상태`
- `[Assets] 일러스트/배경/리소스`
- `[Audio] BGM/SE/보이스`
- `[Save] 저장/불러오기/슬롯`
- `[Log] 스킵/오토/히스토리`
- `[Localization] 텍스트 관리/번역`
- `[Build] 빌드/배포/플랫폼 설정`

### 티켓 제목 규칙
- Story: `[Core] 대화 분기 조건 처리`
- Task: `[UI] 선택지 버튼 컴포넌트 구현`
- Bug: `[Save] 저장 슬롯 로딩 시 Null 오류`

### Description 템플릿(요약)
- 배경/목표
- 요구사항
- 완료 조건(AC)
- 참고(스크린샷/디자인 링크)

## 오늘 결정한 포인트
